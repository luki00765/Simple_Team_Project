Gratulacje! Utworzono projekt

Ta aplikacja zawiera następujące elementy:

  • Strony przykładowe z podstawowymi elementami nawigacyjnymi dla strony głównej, strony z informacjami i strony z danymi kontaktowymi
  • Tworzenie motywów przy użyciu programu ładowania początkowego (Bootstrap)
  • Strona Uwierzytelnianie (jeśli je wybrano) zawiera informacje o sposobie rejestracji i logowania
  • Funkcje platformy ASP.NET zarządzane przy użyciu narzędzia NuGet

Customize app

  • Wprowadzenie do składnika ASP.NET MVC
  • Zmień motyw witryny
  • Dodaj więcej bibliotek przy użyciu narzędzia NuGet
  • Konfiguruj uwierzytelnianie
  • Dostosuj informacje o użytkownikach witryny sieci Web
  • Uzyskaj informacje z serwisów społecznościowych
  • Dodaj usługi HTTP przy użyciu składnika ASP.NET Web API
  • Zabezpiecz interfejs API sieci Web
  • Dodaj aplikacje sieci Web czasu rzeczywistego przy użyciu produktu ASP.NET SignalR
  • Dodaj składniki przy użyciu funkcji szkieletów (Scaffolding)
  • Przetestuj aplikację przy użyciu łącza przeglądarki (Browser Link)
  • Udostępnij projekt
